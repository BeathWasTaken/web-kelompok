# Web Kelompok 15 - 25 Jurnal Referensi PKn

Website daftar 25 jurnal referensi untuk tugas PKN Kelompok 15 dengan tema **Penguatan Wawasan Nusantara di Era Globalisasi & Media Digital**.

## Fitur

- 25 kartu jurnal lengkap dengan link, penulis, dan info SINTA
- Pencarian real-time (judul / penulis / nama jurnal)
- Filter berdasarkan kategori (Wawasan Nusantara, Media Digital, PKn, Nasionalisme, Literasi Digital)
- Dashboard statistik
- Dark mode dengan persistensi via localStorage
- Responsif untuk mobile

## Struktur

Semua dijadikan satu file standalone:

```
index.html   <- HTML + CSS + JS
```

## Deploy ke Vercel

1. Hubungkan repo ini ke Vercel (Import Project).
2. Framework Preset: **Other** (static HTML).
3. Build Command: kosongkan.
4. Output Directory: kosongkan (root).
5. Deploy.

Atau via CLI:

```bash
npm i -g vercel
vercel
```

## Menjalankan Lokal

Cukup buka `index.html` di browser, tidak perlu server.
